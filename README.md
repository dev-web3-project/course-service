# Course Service

Ce microservice gère la partie pédagogique de la plateforme LMS (Cours, Modules, Quiz, Devoirs, etc.).

## Fonctionnalités
- Gestion des cours et modules
- Gestion du matériel pédagogique
- Évaluation (Quiz et soumissions)

## Exécution
Ce service est conçu pour fonctionner au sein de l'infrastructure `core`.
1. Assurez-vous qu'il est cloné dans le dossier `lms-microservices/course-service` du projet Core.
2. Lancez `./run-local.sh` depuis le projet Core.

Le service tournera sur le port `8083`.
